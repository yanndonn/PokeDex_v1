# PokeDex_v1

A PokeDex web application built with React, TypeScript, Tailwind CSS, and Vite.

## Tech Stack

- **React 18** — UI library
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling
- **Vite** — Build tool & dev server

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
